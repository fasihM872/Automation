# Email Templates

Add one HTML email template per niche in this folder:

- `dentists.html`
- `plumber.html`
- `hospital.html`
- `care_homes.html`
- `pharmacy.html`

When you select a niche in the dashboard, the matching file is loaded automatically and sent as the email body.

Use `{business_name}` where the selected Excel/CSV lead's business name should appear. Older templates that use standalone lowercase `musharp` as the sample business name are still supported. The brand text `muSharp` and links like `https://musharp.com` are left unchanged.
